僕が学校の公式の合格体験記に選ばれないのは、選考した教員にとって都合が悪いからだとしか考えられない。ならばこちらは包み隠さずはっきりと書いていこうと思う。

合格までの道のり

僕は六甲に入学した中学一年生の時に、六甲の勉強だけで大学受験を乗り越えようと考えて、なるべく授業を聞くようにし、定期考査を頑張ってきました。中学は20〜30位、高校は10〜20位くらいをキープできました。でも、高二までの模試は全て地方の医学部でもE判定でした。勉強してるからこそ悔しかったし、塾に入ろうかと思ったが、高等進学塾の全額無料の特待である学年7位を取れるまでは学校で頑張ろうと決めたのですが、結局9位しか取れず断念しました。

高二の冬休みから本格的に「受験勉強」を始めようと思い、東進に入って自学自習で頑張ろうと決めました。まずは特定の科目に力を入れ結果を出そうと思い、高三の一学期は物理と数学しかやってませんでした。夏の北大本番レベル模試はC判を出せて、少し自信がつきました。

7月まで部活をしていて、中々勉強時間がとれず、部活後に東進で23時まで勉強していました。この時期は体力的にしんどかったですが、場所でメリハリをつけて切り替えをすることで集中して勉強に取り組むことができたと思います。具体的には、弓道場では部活を、電車と登下校では英語や古文の単語帳を、東進では数学物理の演習をしていました。

引退後、数学に関して日頃の授業プリントや定期考査が簡単に感じたのと、使っていた問題集を四周したのできりがいいと思い、力を注ぐ科目を苦手な英語に変えようと決めました。物理は電磁気分野が疎かだと思ったので夏休み中は続け、物理と英語だけをやってました。

夏休みの半ばに一つの目標として京大実戦を受けました。歯が立たなかったのですが、京大の過去問を何年か分解いたので、得意な物理や数学の良問に多く会えてとても力がつきました。第二回全統記述模試では各科目演習不足から酷い結果でした。

9月は新演習の有機以外は共テ対策全振りで国語・情報・日本史に力を入れました。9月末の全統共テ模試に照準を定めてひたすら暗記を続けましたが、結果は一ヶ月やったとは思えないほど酷く、本当に悔しかった。その影響で第三回全統記述模試では化学以外悲惨な結果でした。

第三回全統共テ模試が終わると、冠ラッシュに打ち勝つべく冠の過去問をやりまくりました。神大オープン・実戦・本番レベル、北大オープン・実戦・本番レベルの合計6個受けました。多く受けるのは本番慣れという意味でいいのですが、振り返りの時間などを考えると多くても4個くらいでよかったと思います。

冠が全部終わる11月頭からは一切二次をやらず共テの特に国語と日本史に全力を注ぎました。共テ直前のパックでは点数もまとまってきて流れを作ったのですが、本番では目も当てられないような点数をとってしまい、共テ後一週間は泣いていました。

共テ後、どこに出願するかを決めるのですが、詳しくは後の志望校選びのところで書きます。結局は金沢大学の医学部に出願することに決めました。共テ自己採点後から出願まで結構長い時間があります。僕は共テの点数から逃げるために関医の過去問を解いていました。旧帝以外の大学の過去問は基本的に一年一日で回せるので早めに過去問切れを起こさないためにもやってよかったと思います。

結果的に関医・神医それぞれ正規で合格できた。これで二次力はその二人に並べるのだと自信がついた。残りの一ヶ月はいい流れで勉強できた。2/25まで首席合格しか頭になかったし、絶対受かる自信があった。

過去問で点数を算出するのが僕は嫌いだったので最後の一年以外は大まかにしか出していない。先生の「過去問を解いて一喜一憂して落ちた」という失敗談を聞いていたので、あくまで苦手分野の確認と時間配分の練習のためだけにやった。自分の性格的にもそれで正解だったと思います。

試験当日は英語→数学→理科だった。過去問の感じでは平均して、数学ほぼ全完・英語8割・物理ほぼ全完・化学7割だったのだが、まさかの数学0完。自分が一番びっくりしたし、冷や汗が止まらなかった。ここで思ったのは、合格よりも、物理と化学で全力を出し切って終わろうというある意味での諦めでした。結果、物理は3題中2完、化学は全完の10分あまりでした。

いつも全完できていた数学で0完は難化とはいえやらかしたのと、共テのビハインドで不合格を確信していました。でもやりきれたという気持ちはあって、後悔はなかった。落ちたときに一人なのはいやだなと思って後輩や友人を呼んで15人くらいで囲って合格発表を見ました。受験番号を見つけて教室が沸いたし、みんなで喜べて本当にうれしかった。人生でよかったことトップ10には入る。

最後まで全力を出し切れてよかったと改めて思いました。受かってみると周りに支えられていることに気づきます。日頃から徳を積めというのはこのありがたみに気づくためなのだと感じました。最後まで伴走してくれた両親・学校の先生方・東進のスタッフ・学校と東進の戦友たちにこの場を借りて感謝申し上げたいと思います。ありがとうございました。

4. 勉強の本質：伸びない原因と打開策

定期考査はとれるけど模試で成績が出せない原因

結論として、模試で成績がとれないのは受験勉強をしていないからです。それを打開するためにすることが、網羅系の参考書を自分で進めることです。日頃の小テストや定期考査の勉強に加えて網羅系参考書をするのは難しいと思います。しかしそれこそが受験勉強であり、早くやったもん勝ちです。

勉強しても伸びない原因

大きな原因は二つあります。勉強のベクトルを誤っている、または勉強の目的を見失っているということです。

勉強のベクトルについて、僕は日本史で細かいところばかり覚えて全体の流れを覚えることができなかったです。共テは全体的な流れを聞く問題が多いのに、努力のベクトルが違っていたのが反省点です。定期的に人に自分の勉強を話したり相談したりすることも大切だと痛感しました。

次は勉強の目的を見失うことです。そもそも勉強の目的は、勉強ができることです。そして自分が掲げた志望大学に合格する学力をつけることです。studyPlusという学習管理アプリを夏休みくらいから使い始めたのですが、途中から勉強時間を稼ぐことが目的になっていて全然身につきませんでした。

理解の質と過去問演習の向き合い方

巨視的な理解（そういうものだと割り切る考え方）と微視的な理解（なんでこうなるのか根本から追求する考え方）の両方が必要です。化学は前者が、物理は後者が多かった印象です。受験はすべてを理解するのではなくて、問題に対して持っている知識を運用する方が効率がいいと思います。

過去問は問題形式の把握と抜けがないかの確認です。そこで合否を見るためではない。点数と睨めっこするのではなく、ケアレスミスで落としたり、基礎ができていなくて取れたはずなのに落とした部分と向き合う時間にあてることが大切です。

常にオーバーキルを目指して

高校生の早い段階で「この科目いらない」「このレベルでいい」とかそういうのはやめましょう。それが気持ちの緩みを生みます。模試が良かったから、共テが良かったからちょっと油断する、これも良くないです。合格発表で合格をつかみ取るまで本気で、確キルを入れる勢いで指しきりましょう。

5. その他

推薦入試・面接

推薦入試について。まず推薦で受かるのは前期で受かる人だけです。次に、すべては共テの点で決まります。面接でそこまで大きな点数の差をつけることはできません。結局は共テの点です。チャレンジする経験も大切ですが、普通に前期の勉強をしていた方が簡単に受かることもあるので、推薦に固執するのは良くないです。

地域枠は、そもそもその大学に学力が及ばない人間が、医学部卒業後9年間を犠牲にすることで医学部に入れてもらう救済措置です。もし本当に地域医療に興味があるとしても、今浪人の一年を回避するために15年間の未来を確定させるのは明らかに危ないです。別に一般で入ってからも地域医療はできるし、やりたいことが変わるかもしれない。落ち着いて長い目で見てください。

面接では、はじめ多くの想定質問を考えて本番に望むことを目標に対策してきましたが、神医では考えていなかった質問が出てきて全然答えられませんでした。その反省を活かして関医と金医では、自分の本当に医師になりたい理由、その大学にこだわる理由を深く考え、自分の軸を固めて臨みました。背筋をただして、堂々とハキハキと受け答えをしましょう。

面接質問リスト（神医・関医・金医）

神医（10分・圧迫）：一般枠ではなく地域枠を志望する理由 / 神戸大学が地域枠で活動している地域を三つ挙げて / 兵庫の地域医療で不足している診療科を言えるだけ / 地域医療に必要なものを2〜3挙げて / 実際に地域医療をするとして、懸念される医療・生活の問題点を二つずつ

関医（7分・優しめ）：関西医科大学を志望する理由 / 理念「慈仁心境」について / 医師として患者の意思を尊重するか治療の正しさをとるか（はっきりどちらか答えるよう求められる） / 弓道をした理由 / 関医と北大両方受かったらどちらに行くか / 手元のプラカードを読んで感想を答える

金医（7分・優しめ）：自己PR1分 / 医師が過酷な待遇でも志す理由 / 最近気になったニュース / 馬の合わない同僚への対処法 / 石川を出たいかどうか

志望校選び

医学部はどこも同じと言いますが、6年間住む場所が決まるので慎重に決めましょう。周りを見ていて、志望校を一個上げることで成績が飛躍的に伸びることが多いです。手を伸ばしたら届きそうな環境が一番です。

本番は結局ボーダー−6%の金沢大学に出願しました。理由は2年前の志望校であることと、二次の配点が高いことにあります。北大を受けたい気持ちもあったのですが、ある時、無理かもって思ってしまった。これを思った時点で負けだと自分の中で思ったから、金沢大学に出願しました。金沢もだいぶマイナススタートで先生たちはネガキャンしてきましたが、共テ後から2次試験まで、金沢大学を無理だと思ったことはない。数学を全完すればいいから。

定期考査と入試結果の関係

定期考査の順位と大学の合格率は相関があると思います。理科や社会は直接的に役に立つことは少ないですが、生きていくための背景知識が多く含まれています。目標を持って物事に取り組む姿勢が受験勉強にも生かされるのです。

受験は総合点勝負

苦手科目を作ってはいけません。本番では得意科目である物理と数学であまり手応えが良くなかったのですが合格することができました。理由は苦手科目の英語と化学がうまくいったからです。苦手やからと腐らず、最後まで弱点と向き合ったおかげだと思っています。物理を80から90にするより化学を60から70にする方が簡単だということ。

経験を無駄にしない

推薦に向けて頑張ってきた高校三年間の成績や面接練習は前期の面接や筆記の自信につながりました。夏まで部活していましたが、それはメンタル面での自信や勉強の息抜きになり「最後まで諦めない姿勢」を会得できました。何かに本気で打ち込むことで得られたものを次に活かしていく姿勢、これが大切です。自分の取り組み次第で得られる者は十二分に変えることができます。何事も全力で取り組みましょう受かる人は隙間時間を100時間かき集めるのではなくて100%かき集めた人です。

6. 各学年へのメッセージ

中学生へ

東大・京大・国医は浪人との戦いです。現役が打ち勝つ正攻法は圧倒的な国数英で逃げ切ることです。高三は共テの社会や理科で忙しく、国数英をやれる時間はないです。だからこそ以下の二つをやってほしい。

まず学習習慣の確立。定期考査や小テストに向けて学習計画を立て、それを遂行していくことです。これをしていくうちに自分に合った効率のよい勉強の仕方が分かってきます。高校生からはこれを模試や受験に応用していきやすくなります。きっと高三の自分が感謝してくれます。

次に読書。本を読むことで活字に対する読むスピードが上がり、すべての教科でその効果を発揮します。周りの国語が得意な友人は皆読書をしていました。楽しんで読書をしましょう。

高一生へ

高一は文理選択前の最後の学年です。国数英の中での苦手科目を中心に網羅系の参考書を進めていきましょう。各科目「穴」を作ってはいけません。今頑張らなかったら落ちます。

ほかの科目は定期考査の勉強だけしていれば十分です。部活や遊び、課外活動にも積極的に参加しましょう。高三の時に推薦を出したいと思った時にも役立つかもしれません。

高二生へ

文理に分かれていよいよ本格的な受験に向けた勉強が始まります。高二からは理科と数学Ⅲの勉強が始まります。学校を信じるならしっかり授業を聞いてください。どちらを信じるにせよ、中途半端にならないことが大切です。おまえについて行って落ちたらしばいたるくらいの勢いで食らいついていきましょう。

文化祭や部活での幹部など責任ある役職に就くことも増えてきます。将来必ず生きてくる経験なので、全力で頑張ってほしいと思います。

高三生へ

数学は演習を毎日続けましょう。英語は単語帳を定期的に見直してください。半年ノータッチにしたら全然読めなくなりました。物理・化学は早急に範囲を終わらせて演習に入ること。分からない知識があったら基礎に返って確認してほしい。国語は短期間で詰め切ることは難しいので早めから慣れること。社会は夏からで十分で、過去問と暗記を交互にやることを強く薦めます。

春（4〜6月）：場所でメリハリをつけて、単語帳系と物理化学の基礎固めに力を入れてください。体育祭は仲間と一つの目標に向かって力を合わせられる、人生でも数少ない場です。全力で取り組んでください。

夏（7〜8月）：まず予定を立てること。理科を中心に演習を積みましょう。夏休みの最後に第二回全統記述模試があります。夏休みの成果の確認として頑張ってください。

秋（9〜12月）：第三回全統記述模試が終わると冠ラッシュがあります。大学の過去問か冠の過去問をフル活用してください。冠が終わったら共テ全振りで行きましょう。苦手科目で何ができないから点数がとれないのか毎回の演習を大切に取り組んでください。

冬（1〜2月）：共テが良くても悪くても合格という二文字を目にするまで学習を止めてはいけません。過去問では特に時間配分と難化易化への対策をしましょう。周りに惑わされることなく、二次試験最終日まで全力で頑張ってください。

最後に

自分が選んだ道を正解にする努力をしてください。

僕は結果として第一志望校にいけなかったが、それに対する後悔はない。それは自分が勉強に全力で取り組んだからだと思う。全力で努力すればそれに対する後悔をすることはない。自分の思い通りの進路でなかったとしても、今からの行動でなんとでも変えられます。腐るのではなく、この進路でよかったと周りに誇れるように行動してください。これを読んでくれた人たちに何か一つでもためになることを伝えられたら嬉しいです。心から応援しています！
